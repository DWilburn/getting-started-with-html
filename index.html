<b>>Basics of Web and HTML</b> Understanding how users, computers, servers, HTTP, and the internet work together

Basics:

HTML – main document type
URLs – how you refer to documents on the web
HTTP – main protocol

The web is a collection of documents: plain text, HTML, images, video, and music. In a browser users request via the internet, information stored on a server. The server responds with the files displayed on the users browser. HTML stands for HyperText Markup Language . Documents contain text or other information and the “markup” is how it is formatted.

Computers interpret what the programmer inputs literally so mistakes can cause major problems.

HTML Elements

HTML documents are made up of Elements. Element types are determined by tags. Tags are used to “markup” the text in numerous ways. Inline elements are simply line by line while Block Elements form an “invisible box” around its contents and add height and length.
